##### Este projeto é uma pequena api criada para fazer um calculo de frete utilizando os dados consumidos da api ViaCep, onde não só é possivel calcular esses fretes, como também salvar tais consultas(cotação). O Banco de dados usado é o banco H2( banco executado em memória).A finalidade desse pequeno projeto é avaliar a capacidade do candidato a executar a tarefa de criação e manutenção desse tipo de apicação
##### Foi utilizado o Swagger para documentação da api:
url para consulta da documentação: http://localhost:[porta definida]/swagger-ui.html
